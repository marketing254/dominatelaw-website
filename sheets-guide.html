Google Sheets Setup &
Form Data Guide

How to connect all Dominate Law forms to Google Sheets and manage incoming leads, community sign-ups, guest applications, and contact messages.

1

Create a new Google Spreadsheet

Go to sheets.google.com and create a new blank spreadsheet. Name it something like Dominate Law — Form Submissions.

The script will automatically create these tabs (worksheets) the first time each form is submitted:

Legal Tools Gate
Timestamp · First Name · Last Name · Email · Role
Free Downloads Gate
Timestamp · First Name · Last Name · Email · Firm Name · Role
Community Join
Timestamp · Source · First Name · Last Name · Email · Role · Practice Area
Guest Speaker
Timestamp · First Name · Last Name · Title · Organization · Email · Phone · Type · Topic · Bio · Links
Contact Us
Timestamp · First Name · Last Name · Email · Phone · Subject · Message
2

Open the Apps Script editor

In your new Google Sheet, click the menu: Extensions → Apps Script. This opens the script editor in a new tab.

Delete any existing code in the editor (usually a default empty function).

3

Paste the script code

Open the file google-apps-script.js (in the same folder as this guide). Copy its entire contents and paste it into the Apps Script editor.

Click the 💾 Save button (or press Ctrl+S). You can name the project Dominate Law Forms.

// File: google-apps-script.js
// Location: dominatelaw-website/google-apps-script.js
// Copy entire file → paste into Apps Script editor
4

Deploy as a Web App

In the Apps Script editor, click Deploy → New deployment.

  • Type: Web app
  • Execute as: Me
  • Who has access: Anyone

Click Deploy. You'll be asked to authorize — click Authorize access and follow the prompts.

⚠️ Important — Authorize Google

You may see a "Google hasn't verified this app" warning. Click Advanced → Go to Dominate Law Forms (unsafe). This is your own script, it's safe.

5

Copy your Web App URL

After deployment, you'll see a Web app URL like this:

Your URL will look like:

https://script.google.com/macros/s/AKfycb.../exec

Copy this URL — you'll need it in the next step.

6

Add the URL to all HTML files

Open each of these HTML files and find the line var DL_SHEETS_URL = 'YOUR_APPS_SCRIPT_URL';. Replace YOUR_APPS_SCRIPT_URL with your actual Web App URL.

  • tools.html — Legal Tools gate form
  • resources.html — Free Downloads gate form
  • community.html — Community join forms (hero + main)
  • guest-speaker.html — Speaker application form
  • contact.html — Contact Us form
// Find this line in each file:
var DL_SHEETS_URL = 'YOUR_APPS_SCRIPT_URL';

// Replace with:
var DL_SHEETS_URL = 'https://script.google.com/macros/s/YOUR_ID/exec';

After saving and pushing to GitHub, every form submission will automatically appear in the correct tab of your Google Sheet within seconds.

7

Managing your form data

Each form submission adds a new row to the correct tab. Here's how to manage it:

  • Filter by date: Use Google Sheets' built-in filter (Data → Create a filter) on the Timestamp column
  • Export leads: File → Download → .csv to export any tab for use in your CRM or email tool
  • Email alerts: In Apps Script, add MailApp.sendEmail('you@firm.com', 'New Lead', ...) inside doPost() to get instant email notifications
  • Avoid duplicates: The script appends every submission — you can add a de-dupe step in Apps Script if needed
  • Re-deploy after edits: Any time you edit the Apps Script code, go to Deploy → Manage deployments → Edit → create a new version and deploy
📌 Tip — Community tab "Source" column

The Community Join tab has a "Source" column — it will say "Hero Form" or "Main Form" so you know which join form on the community page the person used.
