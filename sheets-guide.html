Google Sheets Setup &
Form Data Guide

Connect all Dominate Law forms to Google Sheets and manage incoming leads, community sign-ups, guest applications, and contact messages — all in one place.

1

Create your Google Spreadsheet

Go to sheets.google.com and create a new blank spreadsheet. Name it Dominate Law — Form Submissions.

The script will automatically create and style these tabs the first time each form is submitted:

Legal Tools Gate
Timestamp · First Name · Last Name · Email · Role
Free Downloads Gate
Timestamp · First Name · Last Name · Email · Firm Name · Role
Community Join
Timestamp · Source · First Name · Last Name · Email · Role · Practice Area
Guest Speaker
Timestamp · First Name · Last Name · Title · Organization · Email · Phone · Type · Topic · Bio · Links
Contact Us
Timestamp · First Name · Last Name · Email · Phone · Subject · Message
2

Open the Apps Script editor

Inside your new Google Sheet, click the menu: Extensions → Apps Script. A new tab opens with the script editor.

Delete any existing code in the editor (there's usually a default empty function myFunction(){}).

3

Paste the script code

Copy the entire script below and paste it into the Apps Script editor. Then click the 💾 Save button (or Ctrl+S). Name the project Dominate Law Forms.

Optional: On line 20, set your email address inside the quotes to get instant email alerts for every new submission.

/** * DOMINATE LAW — Google Sheets Form Handler * Paste into Apps Script → Deploy as Web App * * OPTIONAL EMAIL ALERTS: * Set NOTIFY_EMAIL below to receive an instant email for * every new form submission. Leave '' to disable. */ var NOTIFY_EMAIL = ''; // e.g. 'you@yourfirm.com' var SHEET_HEADERS = { 'Legal Tools Gate': ['Timestamp','First Name','Last Name','Email','Role'], 'Free Downloads Gate': ['Timestamp','First Name','Last Name','Email','Firm Name','Role'], 'Community Join': ['Timestamp','Source','First Name','Last Name','Email','Role','Practice Area'], 'Guest Speaker': ['Timestamp','First Name','Last Name','Title','Organization','Email','Phone','Type','Topic','Bio','Links'], 'Contact Us': ['Timestamp','First Name','Last Name','Email','Phone','Subject','Message'] }; function doPost(e) { var output = ContentService.createTextOutput(); output.setMimeType(ContentService.MimeType.JSON); try { var data = JSON.parse(e.postData.contents); var tabName = data.tab; if (!tabName || !SHEET_HEADERS[tabName]) { output.setContent(JSON.stringify({status:'error',message:'Unknown tab: '+tabName})); return output; } var ss = SpreadsheetApp.getActiveSpreadsheet(); var tab = ss.getSheetByName(tabName); if (!tab) { tab = ss.insertSheet(tabName); var hdr = SHEET_HEADERS[tabName]; var r = tab.getRange(1,1,1,hdr.length); r.setValues([hdr]); r.setBackground('#3A0D00').setFontColor('#E8C44A').setFontWeight('bold').setFontSize(11); tab.setFrozenRows(1); } var headers = SHEET_HEADERS[tabName]; var row = headers.map(function(h){ return h==='Timestamp'?new Date():data[h]||''; }); tab.appendRow(row); try { tab.autoResizeColumns(1,headers.length); } catch(ex){} if (NOTIFY_EMAIL) { try { var subject = '📬 New '+tabName+' — Dominate Law'; var body = 'New submission on Dominate Law\nForm: '+tabName+'\nTime: '+new Date().toLocaleString()+'\n\n'; headers.forEach(function(h){ if(h!=='Timestamp') body+=h+': '+(data[h]||'—')+'\n'; }); body += '\nView sheet:\n'+ss.getUrl(); MailApp.sendEmail(NOTIFY_EMAIL,subject,body); } catch(mailErr){} } output.setContent(JSON.stringify({status:'ok'})); } catch(err) { output.setContent(JSON.stringify({status:'error',message:err.toString()})); } return output; } function doGet(e) { return ContentService.createTextOutput('Dominate Law Form Handler — Active ✓'); }
💡 Want email alerts?

Change line 10 from var NOTIFY_EMAIL = ''; to var NOTIFY_EMAIL = 'you@yourfirm.com'; — you'll get a formatted email the moment anyone fills out a form.

4

Deploy as a Web App

In the Apps Script editor, click Deploy → New deployment.

  • Deployment type: Web app
  • Execute as: Me
  • Who has access: Anyone

Click Deploy. You'll be prompted to authorize — click Authorize access and follow the Google sign-in prompts.

⚠️ "Google hasn't verified this app" warning

This is normal for personal scripts. Click Advanced → Go to Dominate Law Forms (unsafe). This is your own script running under your own Google account — it's completely safe.

5

Copy your Web App URL

After deployment, Google shows you a Web app URL. Copy it — it looks like this:

Your URL will look like:

https://script.google.com/macros/s/AKfycb.../exec

Keep this URL safe — you'll paste it into every HTML file in the next step.

6

Add the URL to all HTML files

Open each of these files in your code editor and find the line var DL_SHEETS_URL = 'YOUR_APPS_SCRIPT_URL';. Replace the placeholder with your actual Web App URL.

  • tools.html — Legal Tools gate form
  • resources.html — Free Downloads gate form
  • community.html — Community join forms (hero + main)
  • guest-speaker.html — Speaker application form
  • contact.html — Contact Us form
// Find this line in each file: var DL_SHEETS_URL = 'YOUR_APPS_SCRIPT_URL'; // Replace with your actual URL: var DL_SHEETS_URL = 'https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec';

Save and push to GitHub. Every form submission will appear in the correct tab of your Google Sheet within seconds.

7

Managing your form data

Every submission adds a new row to the correct tab automatically. Here's how to work with your data day-to-day:

Task How to do it
Filter by date Data → Create a filter → click the Timestamp column header → filter by date range
Export to CRM / email tool File → Download → Comma-separated values (.csv) — works for any individual tab
Search for a person Press Ctrl+F (or Cmd+F on Mac) and type their name or email
Mark leads as contacted Add a "Status" column yourself (right-click a column header → Insert column). Use values like: New · Contacted · Qualified · Closed
Get email alerts Set NOTIFY_EMAIL in the Apps Script (Step 3) — you'll get an email for every new submission
After editing the script Deploy → Manage deployments → Edit → select "New version" → Deploy (redeployment required for changes to take effect)
Avoid duplicate entries The script appends every submission. To de-duplicate, use Data → Remove duplicates in Google Sheets, or add a de-dupe check in the Apps Script
📌 Community Join "Source" column

The Community Join tab has a Source column that tells you whether the person signed up via the Hero Form (top of the community page) or the Main Form (bottom section). Use this to see which placement converts better.

📌 Sharing the sheet with your team

Click the Share button in Google Sheets and add team members with Viewer or Editor access. They can view and manage leads without needing to touch any code.
